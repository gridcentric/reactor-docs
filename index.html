Reactor

Reactor is an open-source management tool for cloud-based application delivery. It's the glue that connects loadbalancers to cloud APIs.

If you're looking to move from a static collection of instances to an elastic pool, Reactor can help bridge that gap.

Worried about failure? Reactor can be clustered for horizontal scalability or to achieve high availability and fault tolerance.
